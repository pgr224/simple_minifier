Documentation


Usuage

Note: I added sample.css, sample.js and sample.html into each folder. They are sample files, so please remove it.

  • Download the sim.zip file and extract into your project folder.
  • Move html files into /html/ folder, move css files into /css/ folder and move js files into /js/ folder.
  • Open minifier.php and Minify your files.
  • After you Minify it, the result (Minified Js and Css files) will be inside /minify/ folder and the minifed html files will be in your project root folder.
Folder Structure


Why Simple Minifier is good?

- SM can minify html,css,js file types.

- SM uses the services of https://javascript-minifier.com and https://cssminifier.com.

- SM let user order their css and js files. (e.g You can order style.css, main.js to be the last, so that your custom codes will replace the above files.)

- SM compress all CSS files into a single file to reduce the http requests.

- SM compress all JS files into a single file to reduce the http requests.

- SM add a short comment in minified files, so that user can re-order the minified data.

- SM skips IE comments and sensitive data when minify HTML files.
